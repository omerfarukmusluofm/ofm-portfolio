Hakkımda

Portföyüme hoş geldiniz! Ben tutkulu bir junior web geliştiriciyim ve modern HTML ve CSS teknikleri kullanarak güzel, responsive web siteleri oluşturmaya odaklanıyorum. Yaratıcı tasarımları fonksiyonel web deneyimlerine dönüştürmeyi seviyorum.

Web geliştirmedeki yolculuğum, web sitelerinin nasıl çalıştığına dair merakla başladı ve frontend geliştirme konusunda gerçek bir tutkuya dönüştü. Şu teknolojilerle çalışmaktan keyif alıyorum:

HTML5 CSS3 Responsive Tasarım Flexbox Grid Layout Modern Web
Geliştirme felsefem hakkında daha fazla bilgi

Temiz, anlamsal HTML ve sürdürülebilir CSS yazmaya inanıyorum. Her proje, yeni bir şey öğrenme ve web teknolojileriyle mümkün olanın sınırlarını zorlama fırsatıdır. Özellikle erişilebilirlik ve web sitelerinin herkes için çalışmasını sağlama konularıyla ilgileniyorum.
